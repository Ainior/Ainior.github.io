发表于 2019-06-07 | 评论数： | 阅读次数：

官网地址：https://hexo.io/zh-cn/

环境依赖

  1. 安装 node
  2. 注册 github 新建仓库名必须为 ‘用户名.github.io’
  3. 安装 git

安装命令

使用命令

  • 新建文章 hexo new article_name
  • 新建页面 hexo new page tag
  • 生成代码 hexo generate
  • 提交代码 hexo deploy
  • 实时预览 hexo s -p 4000
  • 清除缓存 hexo clean
  • 查看版本 hexo version

主题插件

  1. 评论插件 Valine
    1. 注册LeanCloud
    2. 创建应用
    3. 自己百度

  3. 后台管理 Hexo-Admin
    后台有点丑就没用了
